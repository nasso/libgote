gt_dispatcher_t* gt_dispatcher_create ( usize_t  n,
  ... 
)

Create a new gt_dispatcher_t with the given gt_system_ts.

If any of the gt_system_t pointers is NULL, other (non-NULL) systems are destroyed with gt_system_destroy, and this function returns NULL. This allows callers to avoid checking for eventual NULL pointers when calling a constructor that might fail:

// Any constructor might fail and return NULL.
gt_dispatcher_t *my_dispatcher = gt_dispatcher_create(
3,
my_physics_system(0, 9.81f),
my_transform_system(),
my_rendering_system()
);
// If any of them does, gt_dispatcher_create will also fail and return NULL.
// No need to bother destroying the other systems: it's done automatically.
if (!my_dispatcher)
printf("Oops!! Something went wrong...\n");

...n gt_system_t pointers to initialize the dispatcher with.
